================================================================================
  FIRST TIME SETUP (OPTIONAL)
================================================================================

  For enhanced features, add API keys to .env file:

  VITE_VIRUSTOTAL_API_KEY=your_key_here
  VITE_WHOISXMLAPI_KEY=your_key_here
  VITE_GOOGLE_SAFE_BROWSING_API_KEY=your_key_here

  Note: Application works without API keys but with limited features

================================================================================
  TROUBLESHOOTING
================================================================================

  Problem: Application won't start
  Solution: Ensure Python 3.8+ is installed
           Run: python --version

  Problem: Import errors
  Solution: Install dependencies
           Run: pip install -r requirements.txt

  Problem: PyQt5 issues
  Solution: Install PyQt5 separately
           Run: pip install PyQt5 --upgrade
